## 📖 Overview
**BasicCode** is an interactive, multimedia-driven learning platform specifically designed to help beginners master C++ programming fundamentals (specifically tailored for course CSC402). 

Moving away from dry, traditional textbook learning, BasicCode transforms abstract programming theories into engaging, bite-sized visual lessons. It provides a hands-on learning experience where students can read, write, and test their understanding all in one seamless web environment.

## ✨ Key Features
* **Bite-Sized Visual Lessons:** Complex C++ concepts broken down into easy-to-digest, visually engaging modules.
* **Real-Time Code Execution:** An integrated code editor that allows learners to write, test, and run C++ snippets directly in the browser.
* **Gamified Quizzes:** Interactive assessments at the end of each module to test knowledge retention.
* **Locked Progression System:** A structured learning path that requires users to master current topics and pass quizzes before unlocking advanced lessons.
* **Responsive UI:** A clean, accessible interface that works smoothly across different screen sizes.

---

## 💻 Tech Stack
This platform is built entirely as a responsive front-end web application using:
* **HTML5:** For semantic content structure.
* **CSS3 & Bootstrap 5:** For modern, responsive styling and layout components.
* **Vanilla JavaScript (ES6+):** For DOM manipulation, quiz logic, progression tracking, and handling the real-time code execution environment.

---

## 🛠️ Getting Started / Installation
Since BasicCode is a front-end application, no complex local server setup is required.
